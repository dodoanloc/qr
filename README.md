# QR Generator

A beautiful, free QR code generator built with Next.js, TypeScript, and Tailwind CSS.

## Features

- ✨ Generate QR codes from any text or URL
- 🎨 Customizable colors (presets + custom picker)
- 📏 Adjustable size (100px - 1000px)
- 🔧 Error correction levels (L, M, Q, H)
- 💾 Download as PNG or SVG
- 📱 Responsive design
- 🌐 Vietnamese interface

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- qrcode library
- Lucide React icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

### Vercel (Recommended)

1. Push code to GitHub
2. Import repo on [Vercel](https://vercel.com)
3. Deploy automatically

### Static Export

The project is configured for static export. Build files are in `dist/` directory.
